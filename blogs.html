Metas a Cumplir

A través de este proyecto que he llevado a cabo como parte de un curso práctico de maquetación en CSS, me gustaría presentar las tecnologías que deseo aprender y mejorar en mi camino para convertirme en una desarrolladora Front-end.
En mi desarrollo profesional, considero fundamental adquirir conocimientos en HTML, CSS, JavaScript, así como en frameworks de JavaScript y control de versiones.

HTML (HyperText Markup Language)

Es el lenguaje de marcado estándar para crear y estructurar el contenido de una página web. Con el dominio de HTML, podré crear la estructura de una página web de manera semántica y optimizada.

CSS

Es el lenguaje de estilos que permite darle vida y diseño a las páginas web. Con CSS, podré controlar la apariencia visual de los elementos, como colores, tipografías, espaciado y disposición.

JavaScript

Con JavaScript, podré agregar interactividad y dinamismo a las páginas web, creando efectos, validaciones de formularios y manipulación de datos en tiempo real.

Control de versiones (Git y GitHub)

El uso de un sistema de control de versiones como Git, podré realizar un seguimiento de mis cambios, colaborar con otros desarrolladores, revertir modificaciones y mantener un historial completo de las versiones del proyecto.

Frameworks de JavaScript

Existen varios frameworks de JavaScript que facilitan el desarrollo frontend, como React, Angular y Vue.js. Estos frameworks proporcionan herramientas y bibliotecas que permiten crear interfaces de usuario interactivas y eficientes.

React

Deseo enfocarme en uno de los frameworks de JavaScript, como React, que me permitirá crear componentes independientes que se actualizan de manera eficiente ya que facilita el desarrollo de aplicaciones web escalables y de alto rendimiento.
